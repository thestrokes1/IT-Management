# IT Management Platform Documentation

Welcome to the IT Management Platform documentation.

## Overview

This platform provides role-based access control for:
- **Tickets** - Issue tracking and assignment
- **Assets** - Asset management and tracking
- **Projects** - Project management
- **Users** - User management with role hierarchies

## Role Hierarchy

| Role | Level |
|------|-------|
| VIEWER | 1 |
| TECHNICIAN | 2 |
| MANAGER | 3 |
| IT_ADMIN | 4 |
| SUPERADMIN | 5 |
